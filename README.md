## Controls

- **W** - Move Up
- **S** - Move Down  
- **A** - Move Left
- **D** - Move Right
- **Yes** (Right button) - Restart game after Game Over

## Game Rules

1. Control the snake using keyboard controls
2. Eat red apples to grow longer and increase your score
3. Avoid colliding with the snake's own body
4. The game ends if the snake hits itself or hits wall.
6. Game speed increases as your score grows.
